LCSK

LiveChat Starter Kit is an ASP.NET C# Live Support / Live Help chat app built on top of SignalR to offer support to website visitors.

View project onGitHub

One folder to copy into your project

Adding LCSK to your existing project is very easy. It's only a small folder (/LCSK/). You may directly install it via NuGet. 

$ Install-Package LCSK
